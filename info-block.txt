Залить репозиторий на GitHub:
Шаг 1. Создать на GitHub репозиторий.
1) заходим на GitHub;
2) в левой части панели видим: Repositories, а рядом - New;
3) нажимаем на New, придумываем название для нашего репозитория, а при желании - и описание;
4) нажимаем на кнопку "create repository"
5) в графе "code" возле SSH написан код - он нам будет в дальнейшем нужен.

Шаг 2. Работа в редакторе.
1) открываем Visual Studio Code; 
2) открываем папку, в которой создаем файлы;
3) открываем терминал - "новый терминал";
4) в открытом терминале пишем возле пути к файлу после значка > git init
5) в левой части панели возле значка "система управления версиями" (второй сверху, похожий на распутье) появляется в кружочке циферка; файлы в открытой папке окрашиваются в зеленый цвет;
6) далее в терминале пишем git add .
7) файлы добавляются
8) далее в терминале пишем git commit -m "какой-то комментарий на английском" 
9) файлы становятся закомиченными;
10) копируем ссылку, о которой речь шла в шаге 1 пункте 5, пишем git remote add origin и вставляем в командную строку терминала; получается примерно так: git remote add origin https://github.com/Julia1986/start-git.git
11) пишем git push -u origin master
